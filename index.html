Friday, May 4th, 2018

We have been having consistent problems with installing Linux virtual machines in Virtual Box for Windows 10. The problem is that Virtual Box for Windows 10 does not seem to allow 64-bit versions of Linux to be installed; but 64-bit versions are all that are available at this point.

On a Mac, when we install Virtual Box, we then download, say, an Ubuntu disk image. We click on "New" on the Virtual Box menu, and we are taken through an install sequence. At the end, we see the VM we created, with a legend such as "powered off."

When we double-click on the machine for the first time, it takes us to another interactive menu, where we have a drop-down list of machine types we can choose from. By default, in Windows 10, no 64-bit options are available, whereas by default in Mac OSX, many 64-bit options are available. We want to make it so that the Windows systems have the same options as the Mac systems.

The issue has been arising in HP machines. On a Microsoft Surface machine we did not experience the same problem.

The first step towards resolving the problem on HP machines with Windows 10, was booting to BIOS. On HP machines, this is accomplished by restarting the machine, then repeatedly jabbing F9 until you see some text saying that it is going to enter BIOS mode. It may have another name for that, like Setup Utility. When you do boot to BIOS, it will look like this (except the instructor and student reflections will be different).

Notice the menu across the top, with the options Main, Security, System Configuration, and Exit. You move between these with the direction arrows. Go right with the direction arrows until you are in System Configuration. Notice the Virtualization Technology is disabled by default

You should see a menu across the bottom indicating what keys to use. The current HP BIOS has F5 or F6 to change setting, so use F5 or F6 to change Virtualization Technology to Enabled.

Use F10 to save and exit. The computer will restart, and you will now have the 64-bit options in Virtual Box.
